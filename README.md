# AI Code Reviewer

An AI-powered code reviewer built with React.js and Node.js.

## Features

* Review source code using AI
* Get code improvement suggestions
* Simple and responsive interface

## Tech Stack

* React.js (Frontend)
* Node.js & Express.js (Backend)

## Run the Project

1. Install dependencies in `frontend` and `backend`.
2. Add your AI API key in the backend `.env` file.
3. Run `npm start` in both folders.
4. Open `http://localhost:3000` in your browser.
